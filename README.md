# Ngrok Link Masker

This tool retrieves the public Ngrok URL and masks it with a public URL using a URL shortener like TinyURL. It works on Kali Linux, Ubuntu, Termux, and Parrot OS.

1. Clone the repository:

   cd ngrok-link-masker
